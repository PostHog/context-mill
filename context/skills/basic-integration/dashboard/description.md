# Create a starter dashboard

Use the PostHog MCP tools to create a dashboard named "Analytics basics (wizard)"
with a few high-signal insights built on the events this integration captures.

- Read the capture step's handoff for the exact event names — use the real names,
  do not invent events that were not instrumented.
- Add up to five insights, favoring conversion funnels and the events that signal
  activation or churn.

Hand off the dashboard URL so the report can link to it.
